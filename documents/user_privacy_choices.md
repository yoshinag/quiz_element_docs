本ページでは、アプリ「元素！覚える！」から収集されるデータの変更・削除、およびデータの使用方法や共有方法を利用者自身が管理する方法をご案内します。

## 1. 収集されるデータの概要

本アプリは利用者の個人情報（氏名、メールアドレス等）を一切収集しません。端末内に保存されるデータおよび広告 SDK が取得するデータの詳細は、[プライバシーポリシー](privacy_policies/quiz_element_basic.md)をご参照ください。

## 2. データの削除

### 2.1 アプリ内データの削除

学習進捗・設定・ログなど、端末内に保存されたすべてのデータは以下の方法で削除できます。

- **アプリのアンインストール**: アプリを削除すると、端末内のすべてのデータが完全に削除されます。
- **アプリのストレージクリア**:
  - iOS: 設定 → 一般 → iPhone ストレージ → 元素！覚える！ → Appを取り除く
  - Android: 設定 → アプリ → 元素！覚える！ → ストレージ → データを消去

### 2.2 広告関連データのリセット

Google AdMob が取得した広告識別子は、端末の設定からリセット・無効化できます。

- **iOS**: 設定 → プライバシーとセキュリティ → トラッキング → 本アプリの追跡を無効化
- **Android**: 設定 → Google → 広告 → 広告 ID をリセット／削除

## 3. 広告の同意設定の変更

本アプリは Google UMP（User Messaging Platform）を使用して広告の同意を管理しています。同意設定は以下の手順で変更できます。

1. アプリを起動する
2. 設定画面を開く
3. 「プライバシーオプション」をタップする
4. Google UMP の同意フォームが表示されるので、希望する設定に変更する

## 4. 広告の追跡制限

端末の OS 設定から、広告の追跡を制限できます。

- **iOS**: 設定 → プライバシーとセキュリティ → トラッキング → 「Appからのトラッキング要求を許可」をオフ
- **Android**: 設定 → Google → 広告 → 「広告のカスタマイズをオプトアウト」をオン

これにより、本アプリを含むすべてのアプリでパーソナライズ広告が制限されます。
