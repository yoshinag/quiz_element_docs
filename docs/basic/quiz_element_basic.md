本ポリシーは、モバイルアプリ「元素！覚える！」（以下「本アプリ」）における利用者情報の取り扱いについて定めるものです。本アプリは 6 歳以上を対象とした教育用アプリであり、子どもの利用を想定しています。したがって、児童のオンラインプライバシー保護に関する各国法令（日本の個人情報保護法、米国 COPPA、EU GDPR、英国 Age Appropriate Design Code 等）に準拠した運用を行います。

## 1. 対象年齢

本アプリは **6 歳以上（6+）** のすべての方を対象としています。小学生以下の子どもも利用することを前提に設計しており、表示される広告もファミリー向け（全年齢対象）のみとなるよう構成しています。

## 2. 収集する情報

本アプリは、利用者の個人を特定する情報（氏名、メールアドレス、電話番号、住所、生年月日など）を **一切収集しません**。

本アプリまたは本アプリが利用する第三者 SDK が取得する可能性のある情報は以下の通りです。

### 2.1 本アプリ自身が取得する情報

| 項目 | 内容 | 保存場所 |
|---|---|---|
| 学習進捗 | クイズの正答状況、達成度 | 端末内（アプリ専用領域） |
| アプリ設定 | テーマ、音量、表示オプション等の設定 | 端末内（SharedPreferences） |
| アプリ内ログ | クラッシュ・エラー調査用の稼働ログ | 端末内（ローカルファイル） |

いずれも**端末内にのみ保存**され、外部サーバーへの送信は行いません。

### 2.2 広告配信のため第三者 SDK が取得する情報

本アプリは Google AdMob を通じて広告を表示します。Google AdMob は以下の情報を取得する場合があります。

- 広告識別子（Android: 広告 ID、iOS: IDFA / IDFV）
- 端末情報（OS バージョン、言語設定、デバイスモデル）
- 概算の位置情報（国・地域レベル、IP アドレスベース）
- 広告の閲覧・クリックに関する情報

ただし本アプリは子ども向けアプリとして AdMob を設定しているため（後述 §4）、**パーソナライズ広告は配信されず**、広告識別子は広告配信の頻度調整や不正防止等の限定的な目的にのみ使用されます。

## 3. 情報の利用目的

取得する情報は以下の目的にのみ使用します。

- 学習機能（進捗の保存、復習機能）の提供
- アプリ設定の保持
- 不具合調査・品質改善
- 広告の表示（子ども向けの非パーソナライズ広告のみ）
- 不正利用・クリック水増しの防止

マーケティング目的での利用、外部への販売、行動ターゲティングは一切行いません。

## 4. 広告について（子ども向け設定）

本アプリは Google AdMob を使用して広告を表示しますが、6 歳以上の子どもが利用することを前提に以下の設定を適用しています。

- **子ども向け扱い（Tag For Child-Directed Treatment）**: 有効
  AdMob に対して本アプリが子ども向けであることを宣言し、COPPA（米国児童オンラインプライバシー保護法）に準拠した広告配信を要求します。
- **同意年齢未満扱い（Tag For Users Under the Age of Consent）**: 有効
  EU GDPR における同意可能年齢未満の扱いとして配信を要求します。
- **最大広告コンテンツレーティング**: **G（全年齢対象）**
  成人向け・暴力的・不適切な表現を含む広告は配信されません。
- **非パーソナライズ広告のみ**: 利用者の閲覧履歴・興味関心に基づく追跡型広告は配信しません。

これらにより、広告識別子は行動ターゲティングには使われず、表示される広告は全年齢に適したコンテンツに限定されます。

## 5. 第三者サービスの利用

本アプリは以下の第三者サービスを利用しています。各サービスのプライバシー方針も併せてご参照ください。

| サービス名 | 目的 | 提供元のプライバシーポリシー |
|---|---|---|
| Google AdMob | 広告配信 | [policies.google.com/privacy](https://policies.google.com/privacy) |
| Google User Messaging Platform (UMP) | 広告同意管理 | [policies.google.com/privacy](https://policies.google.com/privacy) |

## 6. 情報の第三者提供

本アプリは、法令に基づく場合を除き、取得した情報を第三者に提供することはありません。広告配信のために必要な情報は、Google AdMob の運営元である Google LLC が前項のプライバシーポリシーに従って処理します。

## 7. 情報の保存期間

- 端末内に保存される学習進捗・設定・ログは、利用者がアプリをアンインストールするか、アプリ内の設定からデータを削除するまで保持されます。
- Google AdMob が取得する情報の保存期間は、Google のプライバシーポリシーに従います。

## 8. 利用者の権利

利用者（または保護者）は以下の権利を行使できます。

- **データ削除**: アプリをアンインストールすることで、端末内のすべてのデータが削除されます。
- **広告同意の撤回**: アプリ内の設定画面から「プライバシーオプション」を開き、Google UMP 同意フォームを再表示して設定を変更できます。
- **広告識別子のリセット / 追跡制限**:
  - iOS: 設定 → プライバシーとセキュリティ → トラッキング から本アプリの追跡を無効化できます。
  - Android: 設定 → Google → 広告 から広告 ID をリセット・削除できます。

## 9. 保護者の方へ

本アプリは 6 歳以上のお子様が利用することを想定していますが、以下の点にご留意ください。

- 本アプリは個人情報を収集しません。
- 広告は全年齢対象（レーティング G）に限定されますが、広告内容は Google AdMob により動的に配信されるため、すべての広告を事前に個別確認することはできません。
- 広告をタップすると外部の Web ページやストアに遷移する場合があります。誤タップによる購入等を防ぐため、端末の「子ども用設定」や「スクリーンタイム」機能の併用を推奨します。
- お子様の利用状況について懸念がある場合は、下記の問い合わせ先までご連絡ください。
